# Architecture

- stack: TypeScript + Node.js, con empaquetado npm y salida `dist/`.
- modulos: `src/cli.ts` para comandos, `src/docs.ts` para documentación, `src/scaffold.ts` para generación de proyectos, `src/standard/` para DS-v1, `src/launchers/` para agentes y `src/engine/` para ejecución del pipeline.
