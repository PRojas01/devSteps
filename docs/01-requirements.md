# Requirements

- funcionales: CLI instalable, guía paso a paso, inyección de instrucciones para agentes, validación DS-v1, documentación de publicación, integración con DevControl, compatibilidad con Windows/Linux y editores agénticos.
- no funcionales: TypeScript estricto, Node.js 20+, sin secretos embebidos, publicación reproducible, documentación apta para onboarding de principiantes.
